## Key Concepts

Understanding these fundamental concepts will help you get the most out of the Reports system.

### Reports

A **report** is a structured view of your data, often with visualizations. Reports can be:
- **Tabular**: Data displayed in rows and columns
- **Visual**: Charts, graphs, and other visualizations
- **Mixed**: Combining tables and charts
- **Interactive**: Allowing filtering and drill-down

### Dashboards

A **dashboard** is a collection of widgets (reports, KPIs, charts) arranged on a single screen. Think of it as your personalized control panel for monitoring key metrics at a glance.

### Data Sources

The foundation of every report is a **data source**:
- **Contacts**: People in your CRM
- **Organizations**: Companies and groups
- **Opportunities**: Sales pipeline and deals
- **Tasks**: Activities and to-dos
- **Interactions**: Emails, calls, meetings
- **Custom Objects**: Your organization's custom data

### Visualizations

**Visualizations** transform data into graphics:
- **Table**: Traditional rows and columns
- **Bar Chart**: Compare values across categories
- **Line Chart**: Show trends over time
- **Pie/Donut Chart**: Display proportions
- **Scatter Plot**: Reveal correlations
- **Heat Map**: Visualize patterns in matrices
- **Funnel Chart**: Track conversion processes
- **Gauge**: Display single metric progress
- **Number**: Show key statistics
- **Map**: Geographic data visualization

### Filters

**Filters** limit the data shown in your report:
- **Simple Filters**: Quick filtering by common fields
- **Advanced Filters**: Complex logic with AND/OR conditions
- **Date Range Filters**: Relative (last 30 days) or absolute dates
- **Saved Filters**: Reusable filter templates

### KPIs (Key Performance Indicators)

**KPIs** are metrics that matter to your business:
- Track specific numerical goals
- Compare actual vs. target values
- Monitor trends over time
- Receive alerts on threshold breaches

## Navigating the Reports Interface

### Reports Home Page

The Reports home page is your starting point. Here you'll find:

**Top Navigation Bar**
- **Create Report**: Start building a new report
- **Create Dashboard**: Build a custom dashboard
- **Templates**: Browse pre-built report templates
- **Search**: Find existing reports quickly

**Left Sidebar**
- **My Reports**: Reports you've created
- **Shared with Me**: Reports others have shared
- **Favorites**: Reports you've starred
- **Recent**: Recently viewed reports
- **Folders**: Organize reports into categories

**Main Content Area**
- **Featured Dashboards**: Quick access to important dashboards
- **Recent Reports**: Your last 10 viewed reports
- **Report Gallery**: Browse all available reports
- **Quick Stats**: Key metrics at a glance

**[Screenshot: Annotated Reports home page showing all sections]**

### Report View Page

When viewing a report, you'll see:

**Report Header**
- Report name and description
- Last updated timestamp
- Creator information
- Action buttons (Edit, Share, Export, Schedule, Add to Dashboard)

**Report Controls**
- Refresh button
- Date range selector (if applicable)
- Quick filters
- View options (full screen, print mode)

**Report Content**
- Main visualization (chart, table, etc.)
- Interactive controls (zoom, pan, drill-down)
- Data summary statistics

**Report Footer**
- Data source information
- Row count
- Export options
- Pagination (for tables)

**[Screenshot: Annotated Report view page]**

### Dashboard View

Dashboards organize multiple widgets in a grid layout:

**Dashboard Header**
- Dashboard name
- Edit mode toggle
- Auto-refresh settings
- Share and export options

**Widget Grid**
- Drag-and-drop rearrangement (in edit mode)
- Resize widgets by dragging corners
- Each widget has its own controls
- Real-time data updates

**Dashboard Actions**
- Add new widgets
- Remove widgets
- Reset layout
- Save changes
